## Phys 486 Computational Physics

A collection of numerical methods and homework assignments/projects learned and completed during the duration of my Phys 486 Computational Physics course.

### numerical-methods.py
- A compilation of the approximation methods used in the homework assignments.
	- Euler (1st order)
	- Runge-Kutta (4th order)

### radioactive-decay (HW 1) 
- Explores uncertainty in the age of an object based on decay ratio of type A and B nuclei found within the object, using Euler's Method.
